# Email Extractor

A beginner-friendly Python application that extracts email addresses from text files using a modern graphical interface. It is designed to help users quickly process unstructured or semi-structured data, such as university records or mixed text documents, and accurately identify valid email patterns using regular expressions. The application provides an intuitive GUI with features like file browsing and drag-and-drop support, making it easy to use even for those with minimal programming experience.

---

## Features

* Browse file option
* Drag & Drop support
* Modern Dark Theme UI
* Extracts emails using Regular Expressions
* Removes duplicate emails automatically
* Saves results to a new file

---

## Technologies Used

* Python
* Tkinter (GUI)
* tkinterdnd2 (Drag & Drop)
* re (Regular Expressions)

* File Handling

## Installation

1. Clone the repository:

```
git clone https://github.com/your-username/email-extractor.git
```

2. Install required library:

```
pip install tkinterdnd2
```

3. Run the application:

```
python extractor.py
```

---

## How to Use

1. Drag & drop a `.txt` file OR click **Browse**
2. Enter output file name (e.g., `ExtractedEmails.txt`)
3. Click **Extract Emails**
4. Output file will be generated with extracted emails
